# 文字特徴抽出システム - 要件定義

## 基本要件
- 画像をドロップします。 文字が書かれた画像で透過画像です。 
- 文字を書き文字やエフェクトのある文字に加工するために様々な情報を取得したいと思っています。
- 透過画像の文字特徴を抽出・分析するウェブアプリケーション
- OpenCV.jsを使用した画像処理
- 情報密度を最大化したUI設計

## コード要件
- コメントは記述しないこと
- 字下げインデントは使用しないこと 
- 半角スペースは極力使用しないこと
- 非React環境で実装すること
- 簡潔なコードを心がけること

## UI仕様
- 上部：OpenCV読み込み状況と画像ドロップエリアを横並びに配置
- 左側：パラメータ設定パネル
- 中央上部：元画像表示（高さ約100px）
- 中央メイン：処理済み画像を縦1列に表示（各最大高さ約300px）
- 右側：処理ログ表示
- 全体：余白を極小化し、情報密度を上げる

## 処理内容
画像読み込み後、以下の処理を一括で実行すること：
1. エッジ検出
2. 輪郭抽出
3. 骨格化処理
4. 線分検出
5. 曲線検出
6. 線幅分析
7. 連結成分分析

## その他の要件
- 画像はドラッグ＆ドロップで読み込み可能にすること
- 処理ボタンは不要、画像読み込み後に自動処理を行うこと
- OpenCVの読み込み状態を確認してから処理を開始すること
- 処理内容をログとして簡潔に表示すること
