# AI画像生成の進捗表示改善

## 進捗表示のフォーマット

現状のスピナーのみの表示を、以下のフォーマットに変更する:

```
○×3 : 5/20
```

- `○` : スピナー（実行中を示すアニメーション）
- `×3` : キュー内の残りタスク件数（`TaskQueue`の`getWaitingCount()`を利用）
- `5/20` : 現在のサンプリングステップ進捗（ComfyUI WebSocketの`progress`メッセージから取得）

この表示はレイヤーパネルヘッダに配置し、ページ切り替え時もキュー全体の状況が常に把握できる状態を維持する。

## ComfyUI WebSocket進捗の活用

現状`comfyui-util-v2.js`でWebSocketの`progress`メッセージ(`{value, max}`)を無視している。
これをパースして`5/20`のようなステップ表示に反映する。

## レイヤー単位のキャンセルボタン

- 対象パネルへのオーバーレイは行わない（ページ切り替え時に状態が見えなくなるため）
- 代わりに、レイヤーパネルの各レイヤー項目にキャンセルボタンを配置する
- ComfyUIの場合は`DELETE /queue`で該当promptをキャンセルする
- キャンセル後はスピナーとキュー件数を即座に更新する
